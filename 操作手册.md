# 字母下落射击游戏 - 用户操作手册

欢迎使用字母下落射击游戏！本手册将指导您如何开始和操作游戏，以及掌握最新的连击技巧。

## 1. 启动游戏
双击运行目录下的 `game.exe` 文件即可启动游戏窗口。

## 2. 游戏界面说明
游戏窗口分为两个主要区域：
*   **左侧游戏区**：带有黑色边框的白色矩形区域，字母将从这里随机下落。
*   **右侧状态栏**：
    *   **最高记录**：显示历史最高分（深蓝色）。
    *   **当前得分**：本局游戏的实时得分。
    *   **生命值**：显示剩余生命值（初始为10）。
    *   **当前等级**：显示当前的游戏难度等级。
    *   **Combo提示**：当连续击中时，会显示红色的连击倍数。

## 3. 游戏操作

### 3.1 开始游戏
在游戏窗口顶部菜单栏中，点击 **"游戏(G)"** -> **"开始游戏(S)"**。
*   屏幕顶部将出现一个随机颜色的字母开始下落。

### 3.2 消除字母 (核心玩法)
观察屏幕上正在下落的字母，在键盘上按下**对应的字母键**。
*   **不区分大小写**：看到 'A'，按下 'a' 或 'A' 键均可。
*   **击中反馈**：
    *   **成功**：字母消失，播放清脆提示音，积累 Combo 数。
    *   **失败**：播放低沉提示音，Combo 清零，扣除1点生命值。

### 3.3 进阶技巧：连击系统
这是获得高分的关键！
*   **保持连击**：连续无失误地击中字母，Combo 数会不断增加。
*   **得分倍增**：Combo 越高，每个字母的分数越高（公式：`1 + Combo/5`）。
*   **生命回复**：每当你达成 **10连击**（如 x10, x20...），系统会自动回复 **1点生命值**，并播放特殊的奖励音效。
*   **避免失误**：任何一次按错键或让字母落地，都会导致 Combo 立即归零。

### 3.4 暂停与继续
如果在游戏中途需要休息：
*   按下键盘左上角的 **`ESC`** 键。
*   屏幕中间会显示红色的 **"已暂停"** 字样，字母停止下落。
*   再次按下 **`ESC`** 键即可恢复游戏。

### 3.5 游戏结束
以下情况会导致游戏结束 (Game Over)：
*   **生命耗尽**：当生命值降为 0 时（即累计失误 10 次），游戏结束。
*   屏幕将显示 "Game Over!"、最终得分，如果是新纪录还会显示绿色的 "新纪录！" 字样。

## 4. 菜单功能
*   **关于菜单**：点击菜单栏 **"游戏(G)"** -> **"关于(A)"** 可以查看简要的游戏说明。
*   **退出游戏**：点击 **"游戏(G)"** -> **"退出(E)"** 或直接点击窗口右上角的关闭按钮。
