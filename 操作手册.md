# 字母下落射击游戏 - 用户操作手册

欢迎使用字母下落射击游戏！本手册将指导您如何开始游戏，并利用各种机制刷新最高分记录。

## 1. 启动游戏
双击目录下的 `game.exe` 文件即可启动游戏。

## 2. 界面说明
*   **左侧游戏区**：字母下落的区域。
*   **右侧状态栏**：
    *   **最高记录**：显示历史最高得分（深蓝色）。
    *   **当前得分**：本局游戏的累计分数。
    *   **生命值**：初始为 10，字母落地或按错键会扣除。
    *   **全屏炸弹**：当前剩余的炸弹次数。
    *   **Combo**：连击数显示（2连击以上显示）。

## 3. 核心操作

### 3.1 开始与基础玩法
*   通过菜单 **"游戏" -> "开始游戏"** 开启挑战。
*   按下对应的字母键（不区分大小写）消除下落的目标。
*   **自动瞄准**：如果屏幕上有多个相同字母，系统会自动瞄准位置最低的目标。

### 3.2 进阶机制：连击 (Combo)
*   **连续击中**：不间断击中字母会积累 Combo 数。
*   **得分奖励**：Combo 越高，消除单个字母的分数越高（公式：`1 + Combo/5`）。
*   **中断风险**：一旦有字母落地或您按错按键，Combo 会立即清零。

### 3.3 战术技能：全屏炸弹 (BOMB)
*   当屏幕上的字母过于密集时，按下 **`SPACE` (空格)** 键。
*   **效果**：清空当前屏幕所有字母，并获得分数。
*   **限制**：每局仅限 3 次，请在危急时刻使用。

### 3.4 暂停功能
*   按 **`ESC`** 键可随时暂停或恢复游戏。

## 4. 胜负判定
*   **失败条件**：当生命值降为 0 时游戏结束。
*   **结算**：游戏结束时显示最终得分，若打破记录将提示 "新纪录！" 并自动保存。

祝您练习愉快，不断突破最高分！
