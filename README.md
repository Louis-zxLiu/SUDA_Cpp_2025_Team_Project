# 字母下落射击游戏 (Letter Shoot Game)

![C++](https://img.shields.io/badge/language-C++-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

一个基于 Windows API (Win32) 开发的轻量级 C++ 打字练习游戏。无需任何第三方图形库，仅需 C++ 编译器即可编译运行。

## 🎮 游戏简介

屏幕上方会不断随机掉落字母，玩家需要在字母落地前按下键盘上对应的按键来“射击”消除它们。随着得分的增加，游戏难度（下落速度）会逐渐提升。这不仅是一个有趣的小游戏，也是练习盲打和反应速度的好工具。

## ✨ 功能特性

- **原生 Win32 开发**：极小体积，无外部依赖。
- **动态难度**：每得 10 分提升一级，下落速度随等级增加。
- **视觉反馈**：支持 Unicode 中文显示，随机颜色的字母。
- **音效系统**：击中和失误都有不同的 Windows 系统音效反馈。
- **游戏控制**：支持随时暂停 (`ESC` 键)。
- **计分板**：实时显示得分、失误次数及当前等级。

## 🚀 快速开始

### 环境要求
- Windows 操作系统
- C++ 编译器 (推荐 MinGW-w64 或 MSVC)

### 编译指南

如果你使用 MinGW (g++)，请在终端执行以下命令：

```bash
g++ main.cpp -o game.exe -mwindows
```

* `-mwindows` 参数用于链接 Windows GUI 库并隐藏控制台窗口。

### 运行游戏
编译成功后，双击生成的 `game.exe` 即可启动。

## 🕹️ 操作说明

| 按键 | 功能 |
| --- | --- |
| **A-Z** | 对应消除屏幕上的字母 (不区分大小写) |
| **ESC** | 暂停 / 继续游戏 |
| **菜单栏** | 开始游戏 / 查看关于 / 退出 |
