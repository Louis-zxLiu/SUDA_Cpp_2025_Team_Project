# 字母下落射击游戏 (Letter Shoot Game)

![C++](https://img.shields.io/badge/language-C++-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

一个基于 Windows API (Win32) 开发的轻量级 C++ 打字练习游戏。无需任何第三方图形库，仅需 C++ 编译器即可编译运行。

## 🎮 游戏简介

屏幕上方会不断随机掉落字母，玩家需要在字母落地前按下键盘上对应的按键来“射击”消除它们。本项目完美融合了打字练习与射击游戏的乐趣，拥有流畅的动画效果和丰富的反馈系统。

## ✨ 核心功能

*   **🌧️ 多重字母下落 (Multiple Falling Letters)**：
    *   屏幕上支持多个字母同时下落，生成频率与下落速度随得分动态提升。
    *   输入对应字母时，系统会自动锁定并击毁位置最低（最危险）的匹配目标。

*   **🔥 连击系统 (Combo)**：
    *   连续击中字母可积累连击数。
    *   **得分倍率**：连击数越高，单个字母得分越多（公式：`1 + Combo/5`）。
    *   **连击中断**：按错键或字母落地将导致连击立即重置。

*   **✨ 粒子爆炸特效 (Particle Effects)**：
    *   击中字母时触发彩色粒子爆裂效果，具有重力模拟的物理轨迹，视觉反馈强烈。
    *   射击时带有激光引导线。

*   **💣 全屏炸弹大招 (Bomb Skill)**：
    *   按下 **空格键 (SPACE)** 释放全屏炸弹。
    *   瞬间清空屏幕上所有字母并获得相应分数，伴随全屏闪烁与爆炸音效。
    *   每局游戏初始拥有 3 颗炸弹。

*   **🏆 最高分记录 (High Score)**：
    *   游戏自动将历史最高分保存至本地文件 (`highscore.dat`)。
    *   实时显示最高纪录，挑战自我极限。

## 🚀 快速开始

### 环境要求
- Windows 操作系统
- C++ 编译器 (推荐 MinGW-w64 或 MSVC)

### 编译指南

如果你使用 MinGW (g++)，请在终端执行以下命令：

```bash
g++ main.cpp -o game.exe -mwindows
```

* `-mwindows` 参数用于链接 Windows GUI 库并隐藏控制台窗口。

### 运行游戏
编译成功后，双击生成的 `game.exe` 即可启动。

## 🕹️ 操作说明

| 按键 | 功能 |
| --- | --- |
| **A-Z** | 射击消除对应的字母 (不区分大小写，自动瞄准最低目标) |
| **SPACE (空格)** | 释放全屏炸弹 (清空当前所有字母) |
| **ESC** | 暂停 / 继续游戏 |
| **菜单栏** | 开始游戏 / 查看关于 / 退出 |
